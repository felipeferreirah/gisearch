# Gisearch

![](https://i.ibb.co/7XHmCV2/gisearch.png)

It's a react project to make a simple search for an user and repositories using GitHub API.

## First install

Use the package manager [yarn](https://yarnpkg.com) to install.

```bash
yarn install
```

## Usage

```bash
yarn start
```
## Build

```bash
yarn build
```
